1. html -The root element of an HTML document that contains all other HTML elements.
  2. head - Contains meta-information about the HTML document, such as the title and links to external resources like stylesheets and scripts.
  3. title -Sets the title of the HTML document, which appears in the browser's title bar or tab.
  4. link- Specifies external resources like stylesheets, often used for defining the document's style.
  5. body - Contains the main content of the HTML document, including text, images, and other elements visible on the webpage.
  6. p - Represents a paragraph of text.
  7. h - represents heading from h1 -h6
  8. anchor - Creates hyperlinks to other web pages or resources
  9. img - Embeds images into the document.
  10. div - A block-level container often used for grouping and styling elements.
  11. li - Represents a list item
  12. ol - Defines an ordered (numbered) list.
  13. ul - Defines an unordered (bulleted) list.
  14. table - Creates a table for organizing data with rows and columns.
  15. span - An inline container often used for applying styles or scripting to a portion of text.
